# 範例國中 課表查詢系統 (Demo)

這是一個靜態網頁課表查詢系統的範例專案，設計用於展示不需要伺服器後端，即可透過簡單的 CSV 與 JSON 提供快速查詢的現代化、響應式（RWD）網頁介面。

## 專案簡介
本範例僅包含前端網頁檔案與三個班級 (101, 102, 103) 的虛擬測試資料，**不包含任何真實學校隱私資料**。

## 目錄結構
```
timetable-demo/
├── index.html              # 系統入口與前端版面佈局
├── style.css               # 樣式表（含 RWD 與光影特效）
├── app.js                  # 核心邏輯（課表渲染、CSV解析、過濾）
├── config.js               # 系統設定檔（學期切換、網頁標題設定）
├── timetable_sample.csv    # 系統資料表：範例課表 CSV
└── homerooms_sample.json   # 範例班級-導師 對應檔
```

## 執行方式
由於現代瀏覽器的 CORS 安全性限制，不建議直接點擊兩下 `index.html` 開啟。建議透過本地伺服器執行：
```bash
python -m http.server 8080
```
然後在瀏覽器開啟 `http://localhost:8080` 即可檢視並操作。
